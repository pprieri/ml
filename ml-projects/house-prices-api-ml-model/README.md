# Deployment of Machine Learning Models
Created a fully deployed machine learning model, considering CI/CD, into cloud platforms and infrastructure.

> Built a production-ready Regression model using scikit-learn pipelines & OOP
>Created a Python Package using tox and publish it into a private index server
>Defined tests, logging and a CI/CD layer to guarantee reproducibility
> Served the model via REST API using FastAPI
>Used Docker to deploy the API and publish it in  Heroku and AWS ECS
